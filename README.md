# Customer Churn Analysis (Power BI)

## 📊 Overview
This project analyzes customer churn behavior using Power BI, focusing on identifying when users are most likely to churn and how retention evolves across the customer lifecycle.

## 🔍 Key Insights
- Overall churn rate is approximately 26%
- Churn is significantly higher in early-stage users (0–3 months: 56%)
- Churn decreases as customer tenure increases (12+ months: 17%)
- Early engagement and onboarding are critical for improving retention

## 📈 Dashboard Features
- Churn rate KPI
- Customer segmentation by tenure
- Churn distribution (active vs churned users)
- Churn rate comparison across lifecycle stages
